ShelfBaseClass

This item is only intended to be used by the module's authors. Private

Author: ArtemisTheDeer Date: 01/29/2024 Project: roblox-sparkles

Description: Base class for shelves, inherits basic furniture behavior from FurnitureBaseClass

Properties​

PRODUCT_ROW_SIZES​

</>
ShelfBaseClass.PRODUCT_ROW_SIZES: number

The size of props (Depth-wise) in studs based on their classification Props are spaced out based on the position of the last placed item in the row array + the size (Depth) of the prop's class
