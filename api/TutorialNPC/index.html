TutorialNPC

This item only works when running on the client. Client

Author: ArtemisTheDeer Date: 2/13/24 Project: roblox-sparkles

The tutorial NPC is based off of/inherits from the NPC client class

Tutorial NPCs are not controlled by the server, do not have pathfinding/movement enabled (Using humanoids), and can only be manipulated by the client
