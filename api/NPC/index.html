NPC

This item only works when running on the server. Server

Author: ArtemisTheDeer Date: 11/17/2023 Project: roblox-sparkles

Description: NPC server class

A note on replication, NPCs are not rendered on the server NPCs have their paths played back via iterating and waiting the relevant travel distance between the last successfully reported waypoint + the new waypoint

Properties​

WING_DAMPING_RATIO​

</>
NPC.WING_DAMPING_RATIO: number

Damping ratio for wing flap spring

WING_FREQUENCY​

</>
NPC.WING_FREQUENCY: number

Frequency for wing flap spring

WING_FLAP_CYCLE_TIME​

</>
NPC.WING_FLAP_CYCLE_TIME: number

How long it takes (Baseline idle cycle) for the NPC wings to either open or close

WING_IDLE_SPEED_MULTIPLIER​

</>
NPC.WING_IDLE_SPEED_MULTIPLIER: number

How much the flap speed should be adjusted (Number >= 1 = increase in flap speed, number <= 1 = decrease in flap speed) when the NPC is not moving

WING_MOVING_SPEED_MULTIPLIER​

</>
NPC.WING_MOVING_SPEED_MULTIPLIER: number

How much the flap speed should be adjusted (Number >= 1 = increase in flap speed, number <= 1 = decrease in flap speed) when the NPC is moving

DISABLED_HUMANOID_STATES​

</>
NPC.DISABLED_HUMANOID_STATES: table

Key/pair table (Eg. {FallingDown=true}) of humanoid states that should be disabled by default (To help performance)

PATHFINDING_MAX_ATTEMPTS​

</>
NPC.PATHFINDING_MAX_ATTEMPTS: number

How many times the NPC class (server) will attempt to compute a path before rejecting the promise created by NPC.AssignWorldPath

PATHFINDING_REATTEMPT_DELAY​

</>
NPC.PATHFINDING_REATTEMPT_DELAY: number

How long to wait before re-attempting to build a path after the last path build attempt failed

NPC_PATH_COSTS​

</>
NPC.NPC_PATH_COSTS: table

Key/pair table (Eg. {StoreFurniture=math.huge}) of pathfinding costs for all NPCs See here for more info: https://create.roblox.com/docs/characters/pathfinding#working-with-regions
