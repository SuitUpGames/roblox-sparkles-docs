TemplateController

This item only works when running on the client. Client

Author: Daniel S. Date: 11/28/2023 Project: roblox-sparkles

Description: Handles this particular gui

Setup Instructions:

- Make sure in Assets > UI > Screens, ScreenGui instance or Frame can be found
	that has the same name as this file without the controller.

- Set the DEFAULT_HIDDEN_POSITION (a variable in this script)

- If this is a ScreenGui, then it has to have a Frame inside it
- If this is a Frame, no need to do anything else
