ShopFloor

This item only works when running on the server. Server

Author: Maxim Peng Project: roblox-sparkles

Represents one floor of a shop

Properties​

GRID_UNIT​

</>
ShopFloor.GRID_UNIT: number

Grid unit for snapping prop placement

MAX_UNOCCUPIED_BUILDING_HEIGHT​

</>
ShopFloor.MAX_UNOCCUPIED_BUILDING_HEIGHT: number?

(Optional) How many floors high a building can be (Not including the apartment on top) if nobody owns that store plot. If no number is defined, will default to 1

CFRAME_DECIMAL_PRECISION​

</>
ShopFloor.CFRAME_DECIMAL_PRECISION: number

The decimal place to round saved CFrame position/rotation to when encoding to the datastore

Eg.

1: .1 2: .01 3: .001 4: .0001
