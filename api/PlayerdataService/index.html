PlayerdataService

Author: serverOptimist & ArtemisTheDeer Date: 11/15/2023 Project: Sparkles

Description: Rewrite of serverOptimist PlayerdataService module

Properties​

VERSION​

</>
PlayerdataService.VERSION: number

A number that defines the data version for better QoL (easy resetting).

STORE_NAME​

</>
PlayerdataService.STORE_NAME: string

The datastore to use with profileservice for storing playerdata

DATA_PREFIX​

</>
PlayerdataService.DATA_PREFIX: string

The prefix to amend to the key used for saving playerdata (Eg. "Playerdata_123")

DATA_LOAD_RETRIES​

</>
PlayerdataService.DATA_LOAD_RETRIES: number

The maximum amount of times to try to load a player's data (On joining the game) before rejecting the promise associated w/it

DATA_LOAD_RETRY_DELAY​

</>
PlayerdataService.DATA_LOAD_RETRY_DELAY: number

How long to wait between failed attempts with loading a player's data (On joining the game) before retrying

LOAD_PLAYERDATA_IN_STUDIO​

</>
PlayerdataService.LOAD_PLAYERDATA_IN_STUDIO: boolean

Boolean that determines whether player save profiles should be loaded while in a Roblox studio session If true, playerdata will load in studio. If false, playerdata will not be loaded in studio
