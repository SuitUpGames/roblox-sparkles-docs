PlayerMovementService

Author: ArtemisTheDeer Date: 01/23/2024 Project: roblox-sparkles

Description: Service that manages movement (Flight) of players

Properties​

MAX_ROLL​

</>
PlayerMovementService.MAX_ROLL: number

The max roll (In radians) for the player's flight movement to roll side-to-side based on their movement

BASE_PITCH​

</>
PlayerMovementService.BASE_PITCH: number

The minimum pitch (In negative radians) of the character when in flight mode

MAX_PITCH​

</>
PlayerMovementService.MAX_PITCH: number

The max pitch (In negative radians) of the character when in flight mode

VERTICAL_BOB_OFFSET​

</>
PlayerMovementService.VERTICAL_BOB_OFFSET: number

The max height adjustment on the Y axis (Divided by 2 for each direction) for how far a player should float beyond their current position

BASE_FLIGHT_HEIGHT​

</>
PlayerMovementService.BASE_FLIGHT_HEIGHT: number

How far above the ground (In studs) the player should be at the minimum

FLIGHT_SPEED_MULTIPLIER​

</>
PlayerMovementService.FLIGHT_SPEED_MULTIPLIER: number

The maximum flight speed multiplied relative to StarterPlayer.CharacterWalkSpeed

VERTICAL_FLIGHT_SPEED_MULTIPLIER​

</>
PlayerMovementService.VERTICAL_FLIGHT_SPEED_MULTIPLIER: number

The maximum vertical flight speed multiplied relative to StarterPlayer.CharacterWalkSpeed

VERTICAL_FLIGHT_DEADZONE_CAMERA_ANGLE​

</>
PlayerMovementService.VERTICAL_FLIGHT_DEADZONE_CAMERA_ANGLE: number

The minimum angle (+/-) (In degrees, converted to radians) needed to adjust the desired altitude

VERTICAL_FLIGHT_MAX_CAMERA_ANGLE​

</>
PlayerMovementService.VERTICAL_FLIGHT_MAX_CAMERA_ANGLE: number

The maximum angle (+/-) (In degrees, converted to radians) for adjusting the camera's angle
